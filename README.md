# Online Lodging Marketplace for Short Term Rentals(Go Rental) | MERN Full stack project with Restful Api

Go Rental Project is a travel planning,finding room, lodge, villa, by local areas. •
Implementing a Search, booking system, authentication, authorization, and client/server-side validations using cookies and session management.

# Technology Using

•	Utilized HTML, CSS, JavaScript, Bootstrap, Node.js, Express.js, MongoDB, Implemented CRUD operations and EJS templates for responsive UI.
